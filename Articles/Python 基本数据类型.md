Python 基本数据类型
===================

Python 基本数据类型:
- Number 数字
- String 字符串

-----------

- int
- float
- bool
- complex

常用函数：
- abs()
- divmod()
- pow()
- round()
- sum()
- hex()
- oct()
- chr()
- ord()
